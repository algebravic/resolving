
\section{Symmetries}
\label{sec:symm}
\begin{frame}{Problem Symmetries}
  \begin{itemize}
  \item In a search problem, symmetric solutions slow things down.
  \item Need to ``break'' symmetries.
  \end{itemize}
\end{frame}
\begin{frame}{Hypercube}
  \begin{block}{Permutations}
    An $n \times n$ \emph{permutation matrix}: entries in 0/1 with
    exactly one 1 in every row and column.  Typical: $P$

    An $n \times n$ \emph{signed permutation matrix}: entries in
    $0/\pm1$ with exactly one nonzero in every row and column. Typical:
    $S$. 
  \end{block}
  \begin{itemize}
  \item If $x$ has entries in $0,\pm 1$, so does $S x$, where $S$ is a
    signed permutation.
  \item $A$: binary matrix, so is $P A$
  \item $A$: bipolar matrix then so is $S A$.
  \item Lots of equivalences ($=$ symmetries).
  \end{itemize}
\end{frame}
